# 1. Objetivo de la reunión

Presentar los avances del módulo de mapa interactivo para la gestión de clientes, ubicaciones e inventario, así como validar la funcionalidad CRUD implementada durante el sprint.

# 2. Avances Presentados

### Mapa Interactivo

Se desarrolló una primera versión funcional del mapa interactivo para la visualización de ubicaciones asociadas a clientes.

Funcionalidades implementadas:

* Registro de nuevas ubicaciones.
* Edición de ubicaciones existentes.
* Eliminación de ubicaciones.
* Visualización de ubicaciones en el mapa.
* Asociación de ubicaciones con clientes.

### Gestión de Clientes

Se implementó el CRUD completo de clientes:

* Crear cliente.
* Consultar cliente.
* Editar cliente.
* Eliminar cliente.

### Gestión de Productos

Se implementó el CRUD de productos:

* Alta de productos.
* Consulta de productos.
* Modificación de productos.
* Eliminación de productos.

### Asignación de Inventario

Se implementó la capacidad de:

* Agregar productos a una ubicación.
* Relacionar productos con clientes.
* Consultar inventario asociado a cada ubicación.

# 3. Retroalimentación del Scrum Master

* Validar el funcionamiento del mapa con una mayor cantidad de registros.
* Mejorar la relación entre clientes y ubicaciones.
* Revisar oportunidades de normalización de la base de datos.
* Continuar integrando los módulos de inventario y distribución.
