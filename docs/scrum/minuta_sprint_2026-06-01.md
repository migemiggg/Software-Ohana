# MINUTA DE REUNIÓN SCRUM

## Despliegue y Pruebas del Sistema

**Proyecto:** Sistema de Gestión de Inventario y Distribución Ohana
**Fecha:** 01 de junio de 2026
**Scrum Master:** Maestro de la materia
**Equipo de Desarrollo:** Integrantes del proyecto
**Tipo de reunión:** Seguimiento de Sprint

# 1. Objetivo de la reunión

Presentar el despliegue inicial del sistema en ambiente productivo y coordinar las actividades de prueba con personal de Ohana.

# 2. Actividades Realizadas

### Puesta en Línea

Se realizó el despliegue del sistema utilizando la plataforma Render.

Objetivos alcanzados:

* Publicación del sistema en línea.
* Verificación de acceso remoto.
* Validación de conectividad.
* Pruebas iniciales de funcionamiento.

### Gestión de Usuarios

Se generaron credenciales para pruebas del sistema.

Tipos de usuarios creados:

* Administradores.
* Empleados.
* Testers.

Para cada usuario se configuró:

* Nombre de usuario.
* Contraseña inicial.
* Nivel de acceso correspondiente.

### Validación Funcional

Se realizaron pruebas sobre:

* Inicio de sesión.
* Gestión de clientes.
* Gestión de productos.
* Gestión de ubicaciones.
* Operación del mapa interactivo.
* Consulta de inventario.

# 3. Resultados

* El sistema quedó disponible para acceso remoto.
* Los usuarios de prueba pudieron acceder correctamente.
* Se identificaron oportunidades de mejora en inventario y recetas para futuros sprints.
* Se confirmó el correcto funcionamiento de los módulos principales.

# 4. Retroalimentación del Scrum Master

* Continuar validando procesos de inventario.
* Mejorar trazabilidad de movimientos.
* Incorporar métricas de productos terminados.
* Optimizar la experiencia de usuario en búsquedas y consultas.

# 5. Planeación Próxima Revisión

1. Revisar inventario de productos terminados.
2. Mejorar integración entre recetas e inventario.
3. Incorporar movimientos de productos.
4. Revisar normalización de tablas.
5. Integrar pedidos con mapa y ubicaciones.

# 6. Acuerdos

* Se aprobó el despliegue realizado en Render.
* Se autorizó continuar las pruebas con usuarios de Ohana.
* Las observaciones encontradas serán presentadas en la siguiente Sprint Review.
